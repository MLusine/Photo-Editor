# Photo Editor App

This is a full-stack photo editor app built with:

- React (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)

## Features

- Upload photos
- Edit (crop, resize, pixelate, grayscale, remove background)
- Save to gallery
